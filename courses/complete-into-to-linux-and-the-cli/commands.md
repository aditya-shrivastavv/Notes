# Commands

## Basic

- `pwd`
  - print current working directory
- `clear`
  - clear the terminal
- `anything --help`
  - get help for a command
- `cd`
  - change directory
  - `cd ..`
    - go up one directory
  - `cd ~`
    - go to home directory
  - `cd /`
    - go to root directory
  - `cd -`
    - go to previous directory
  - slash compares to root directory, for relative path, use `cd **name**`
